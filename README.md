## ✨ 简介

使用`React18`,`Typescript`,`Vite`,`Antd5.0`等主流技术开发的开箱即用的中后台前端项目，`Vite`实现自动生成路由，`react-redux`状态管理，支持虚拟滚动表格，`UnoCss`开发样式。

## 💻 安装使用

- 获取项目

```bash
git clone https://github.com/ipinker/ipink-admin.git
```

- 进入目录

```bash
cd ipink-admin
```

- 安装依赖
```bash
yarn install
```

- 运行

```bash
yarn dev
```

- 打包

```bash
yarn build
```
